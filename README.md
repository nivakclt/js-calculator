# JavaScript Calculator

A responsive calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a clean, modern, and user-friendly interface.

## 🚀 Live Demo
https://nivakclt.github.io/js-calculator/

## ✨ Features
- Perform addition, subtraction, multiplication, and division
- Responsive design for desktop and mobile devices
- Modern and clean user interface
- Interactive button effects
- Fast and easy to use

## 🛠️ Built With
- HTML5
- CSS3
- JavaScript

## 📂 Installation

1. Clone the repository:
```bash
git clone https://github.com/nivakclt/js-calculator.git
```

2. Open `index.html` in your browser.
